# Nuhex

Nuhex is a production-ready Nuxt application built with a Hexagonal Architecture (Ports & Adapters) and Domain-Driven Design principles.

The goal of this project is to demonstrate how to structure a modern frontend application with clear separation of concerns, scalability, and maintainability in mind.

## Tech Stack

- Nuxt (latest stable)
- Vue 3 + Composition API
- TypeScript (strict)
- TanStack Query (server state)
- TailwindCSS (UI)
- Vercel (deployment)

## Architecture

This project follows:

- Hexagonal Architecture (Domain, Application, Infrastructure, UI)
- DDD-inspired modular structure (feature-based modules)
- Clean Code principles
- Doc-as-Code for documenting architectural decisions

## Purpose

Nuhex is both:

- A learning project to master Nuxt in a professional context
- A real-world application designed to be deployed and used in production
